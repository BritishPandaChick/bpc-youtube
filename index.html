What is BritishPandaChick?

BritishPandaChick is a coding vlog that describes my experiences in tech, lessons I've learned, and all the victories I've accomplished throughout my coding journey.

  • Share what I've learned as a frontend web developer by reviewing coding concepts and expanding on topics I've written about on my blogs, Code Newbie, and DEV.
  • Help new developers get started in tech by sharing takeaways I've learned from webinars, providing resources that can help developers learn how to code, and coding through tutorials so new developers can code along with me.

If you are interested in learning how to code or just getting started on your own coding journey, subscribe and code along with me.

Channel Rules

BritishPandaChick is a positive space where everyone regardless of where they are in their coding journeys can come together to talk about code, share struggles, celebrate successes, and learn together. If you want to comment on the videos in the channel, here are the 3 rules everyone needs to follow:

  • Be nice.
  • Be honest.
  • Be helpful.
